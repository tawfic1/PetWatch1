# PetWatch

PetWatch is a PHP MVC web application designed for reporting and tracking missing and found pets.

## Features

* User authentication and sessions
* Owner-only pet management
* AJAX/JSON APIs
* Interactive Leaflet maps
* Pet sightings with coordinates
* Search and filtering
* Pagination
* Secure database interaction using PDO prepared statements

## Technologies

* PHP
* MySQL
* JavaScript
* AJAX
* Leaflet.js
* HTML/CSS
* MVC Architecture
